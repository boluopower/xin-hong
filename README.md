# Xin Hong Data


Put your "新红网红数据" folder under this repo like this:

    /xin-hong
        /新红网红数据
            1美妆.xlsx
            2美容个护.xlsx
            3鞋包潮玩.xlsx
            4穿搭打扮.xlsx
            5美食.xlsx
            6母婴育儿.xlsx
            ...
        README.md
        .gitignore
        main.js
        package.json

Then install necessary packages from Terminal:

    npm install

Then run main.js from Terminal:

    node main.js
